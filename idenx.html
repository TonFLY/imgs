🔨 Guia Básico de Upgrade

Neste guia você aprenderá como funcionam os upgrades de equipamentos no Flyff, quais materiais utilizar, como funciona o sistema de Upgrade Garantido e também como perfurar seus equipamentos.

📍 NPC BoboChan

Em Flaris encontramos o nosso conhecido BoboChan, o ferreiro responsável por praticamente todos os upgrades de equipamentos do jogo.

Ao conversar com ele você encontrará diversas opções.

Todas essas opções são responsáveis pelos aprimoramentos dos seus equipamentos.

⚔️ Aprimoramento de Equipamentos

Ao selecionar Aprimoramento de Equipamento, será aberta a tela abaixo.

Tipos de Armas

Existem dois tipos de armas no Flyff:

  • Armas Normais
  • Armas Ultimate

Cada uma utiliza materiais diferentes para realizar upgrades.

🗡️ Upgrade de Armas Normais

Para aprimorar armas comuns serão necessários:

  • ✔ Sunstone
  • ✔ Pergaminho de Proteção S

✨ Upgrade de Armas Ultimate

Armas Ultimate utilizam materiais diferentes:

  • ✔ Oricalkum Brilhante
  • ✔ Pergaminho de Proteção X

💎 Como criar Oricalkum Brilhante

Ao lado do BoboChan está a NPC Peach, responsável pela criação dos materiais especiais e venda dos Pergaminhos de Proteção.

Selecione a opção:

Criar Oricalkum

Depois selecione:

Criar Oricalkum Brilhante

Importante

Para produzir 10 Oricalkum Brilhante são necessários:
  • 50 Sunstones
  • 50 Moonstones

📈 Entendendo a tela de Upgrade

Na tela de upgrade existem três informações muito importantes:

  • Meta de Upgrade
    Mostra qual nível de upgrade você deseja alcançar.

  • Upgrade Atual
    Mostra o nível atual do equipamento.

  • Upgrade Garantido
    Exibe quantas tentativas já foram realizadas e quantas faltam para o sucesso garantido.

⭐ Sistema de Upgrade Garantido

Um dos grandes diferenciais deste servidor é o sistema de Upgrade Garantido.

Cada tentativa malsucedida aumenta o contador.

No exemplo acima, o equipamento garante sucesso em até 5 tentativas.

Caso todas as tentativas anteriores falhem, a quinta será 100% garantida.

Este sistema existe para praticamente todos os equipamentos que podem receber upgrade.

🕳 Perfuração de Equipamentos

A perfuração funciona praticamente da mesma forma que o upgrade.

Os materiais utilizados são:

  • Moonstone
  • Pergaminho de Proteção G

É possível perfurar:

  • Armas
  • Escudos
  • Livros
  • Suits

🛡 Perfuração de Suits

Os Suits possuem um comportamento diferente.

Eles podem receber até 4 perfurações.

Neles podem ser inseridas cartas de:

  • Ataque (%)
  • HP (%)
  • EXP (%)
  • Mana (%)

⚔ Perfuração de Armas e Escudos

Armas, Escudos e Livros funcionam da mesma forma.

Eles podem receber até 10 perfurações.

Neles utilizamos cartas de atributos fixos (Flat), como:

  • STR
  • DEX
  • STA
  • INT
  • Ataque
  • Defesa

🎉 Conclusão

Agora você já conhece os principais sistemas de upgrade do servidor:

  • Upgrade de Equipamentos
  • Upgrade de Armas Ultimate
  • Criação de Oricalkum Brilhante
  • Upgrade Garantido
  • Perfuração
  • Cartas para Suits
  • Cartas para Armas e Escudos

Explore o BoboChan e a Peach para fortalecer seus equipamentos e evoluir seu personagem cada vez mais.
